# NoteDeck Branding

対外的な**求心力**の方針をまとめたドキュメント。内部向けの機能ポジショニングは [STRATEGY.md](STRATEGY.md)、UX 設計思想は [DESIGN.md](DESIGN.md) を参照。

STRATEGY.md が「NoteDeck とは何か（IDE）」を定義するのに対し、本書は「**それをどの順番で、誰に、どう伝えるか**」を定義する。両者は矛盾しない — IDE という正体は変えず、伝える順番だけを設計する。

## 課題 — アイデンティティが、そのまま参入障壁になっている

NoteDeck の差別化要因（IDE・AI 連携・プロトコル可視化・自己拡張）は、そのまま moat（堀）であり、同時に壁でもある。

- moat を捨てれば Misskey 廃人・開発者が離れる
- moat を先頭に置けば Misskey 一般ユーザーが入口で帰る

旧ランディングページは後者だった。H1 が `Misskey Integrated Deck Environment` という純粋なジャーゴンで始まり、2 画面目で `なぜ「IDE」を名乗るのか` に突入していた。一般ユーザーは 1 秒で「自分向けではない」と判断する。

**解決は「薄めること」ではなく「順番を変えること」。** 難しい話を捨てるのではなく、簡単な話の下に沈める。

## ブランドの軸

> **あなたの Misskey が棲む、デッキ。**

「IDE」「統合環境」という*説明*をやめ、**棲む（dwell）**という*感情*を軸に置く。これは二層に効く：

- **一般ユーザー** — 「自分の居場所」という親しみ
- **廃人・開発者** — デッキに常駐する AI / HEARTBEAT / 自己拡張が「デッキに棲む存在」として深層で回収される伏線になる

ブランド名「NoteDeck」自体は親しみやすく無傷。直すのは `Integrated Deck Environment` という*位置づけコピー*だけ。

## iceberg 構造 — 各層が読者を自己選択でフィルタする

ランディングページもアプリ訴求も、上から下へ「ポップ → 廃人ご褒美 → 開発者」と深くなる単一の氷山として構成する。各層が対象読者を自己選択でフィルタするので、誰も入口で帰らない。

| 層 | 対象 | 内容 | 役割 |
|---|---|---|---|
| **上層・ポップ** | Misskey 一般ユーザー | マスコット + 感情的な一行 + スクショ + 3 本柱（速さ / オフライン / 全鯖統合）+ ダウンロード | 一般層はここで DL して離脱して良い |
| **中層・廃人ご褒美** | Misskey 廃人 | デッキ UI / カラム 16 種 / Boss Key / MFM / プラグイン。ここで「実はこれ "IDE" なんです」リビール | スクロールした人が「ただのクライアントじゃない」と気づく |
| **深層・貢献** | 開発者 | なぜ IDE を名乗るのか / プラットフォーム / アーキテクチャ / Hackable | 怖い話はすべて DL CTA より下。スクロールした人だけが見る |

### 「IDE」という単語の扱い

「IDE」は殺さない — 廃人はこれを愛するし、これが正体。だが **H1 から降格**させ、**中層のリビール**として初出させる。「実は、これ "IDE" なんです」という*ご褒美の発見*にすることで、単語を後から効かせる。

ダウンロード CTA は上層（中盤）に前倒しで置く。一般ユーザーが深層の専門的な話に触れる前に行動を完了できるようにする。

## マスコット — デッキに棲む小さな相棒

ロゴ（緑グラデーションの 3 本柱が象る「m」= Misskey の m かつデッキの 3 カラム）の**間や上にちょこんと棲む小さな生き物**。

- **人型キャラではなくクリーチャー / 精霊系** — 描きやすく、性別・年齢を超えて万人に親しみやすく、「棲む」というブランド軸と一致する
- マスコットが**「ポップな表層」と「深い意味」の橋**になる
  - 表層：かわいい相棒 → 一般ユーザーの求心力
  - 深層：その正体は、デッキに常駐して投稿・自己拡張・HEARTBEAT をこなす AI / 自動化 → 廃人へのご褒美
- ロゴの緑（Misskey グリーン）をそのまま継承

## Hero コピー

H1 から `Misskey Integrated Deck Environment` を廃止。タグラインは平易な日本語で。

候補：

- **全部のサーバーが、ひとつの画面に。** — 最も平易、一般ユーザーが一瞬で価値を理解
- **Misskey、ぜんぶまとめて。** — 最もポップ
- **あなたの Misskey が棲む、デッキ。** — ブランドの軸そのもの

ダウンロードボタンは Hero 直下に 1 個だけ置く。

## トーン

[DESIGN.md](DESIGN.md) の「User Respect over Engagement Maximization」に従う。

- ポップ = ドーパミン誘発ではなく、**温かい・親しみやすい**の意味
- 親しみやすさはマスコットの語り口・丸み・余白で出す
- 煽り・FOMO・誇張は使わない（ダークパターン不採用方針と一貫）
